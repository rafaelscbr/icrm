Política de Privacidade — iCRM

Última atualização: 10 de maio de 2026

O iCRM é um sistema de gerenciamento de relacionamento com clientes de uso interno, desenvolvido e operado por Rafael Alves. Esta política descreve como os dados são coletados e utilizados no contexto da integração com a plataforma Meta (Facebook/Instagram).

1. Dados Coletados

Por meio da integração com o Meta Lead Ads, o iCRM pode receber os seguintes dados fornecidos voluntariamente por usuários em formulários de anúncios:

  • Nome completo
  • Número de telefone
  • Endereço de e-mail
  • Informações adicionais preenchidas no formulário do anúncio

2. Finalidade do Uso dos Dados

Os dados coletados são utilizados exclusivamente para contato comercial relacionado ao interesse demonstrado pelo usuário no anúncio, como apresentação de imóveis, envio de propostas e acompanhamento de negociações.

3. Armazenamento e Segurança

Os dados são armazenados em banco de dados seguro (Supabase/PostgreSQL) com acesso restrito ao operador do sistema. Não são compartilhados com terceiros.

4. Retenção de Dados

Os dados são mantidos pelo período necessário para a finalidade comercial ou enquanto o usuário não solicitar a exclusão.

5. Direitos do Usuário

O usuário pode solicitar a exclusão ou correção de seus dados a qualquer momento entrando em contato pelo e-mail abaixo.
